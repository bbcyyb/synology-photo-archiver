# 变更: 创建群晖照片归档程序

## 为什么
为了自动化地备份和归档存储在群晖 NAS 上的照片，确保数据安全，并简化手动操作。

## 变更内容
- 创建一个全新的 Python 应用程序，用于自动发现、打包、加密和归档照片。
- 程序将作为一项新的`photo-archiver`（照片归档器）功能被添加到项目中。
- 关键功能包括：
  - **可配置性**: 所有关键路径（源、目标）和凭据（密码）均可通过配置文件进行管理。
  - **幂等性**: 多次运行程序不会产生重复的归档或副作用。
  - **自动化**: 可通过群晖的任务计划程序进行定时调用。

## 影响
- **受影响的规格**: 无。这是一个全新的功能。
- **新增的规格**: `specs/photo-archiver/spec.md` 将被创建。
- **受影响的代码**: 无。所有代码都是新增的。
