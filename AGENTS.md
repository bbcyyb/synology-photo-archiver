# OpenSpec 说明

这些说明是为在这个项目中工作的人工智能助手准备的。

当请求满足以下条件时，请始终打开 `@/openspec/AGENTS.md`：
- 提及计划或提案（例如：提案、规格、变更、计划等词语）
- 引入新功能、重大变更、架构调整或重要的性能/安全工作
- 听起来模棱两可，在编码前需要权威的规格说明

使用 `@/openspec/AGENTS.md` 来学习：
- 如何创建和应用变更提案
- 规格格式和约定
- 项目结构和指南

请保留此受管块，以便 'openspec update' 可以刷新这些说明。
